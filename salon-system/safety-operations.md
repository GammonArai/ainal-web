# 🛡️ トラブル回避・安全運用ガイド

## 🚨 重要な安全対策

### 1. 身元確認の徹底
```
【初回必須確認事項】
✅ 身分証明書の写真確認（当日持参）
✅ Googleフォームの実名記入
✅ 電話番号の有効性確認
✅ X アカウントの妥当性チェック
✅ 過去のトラブル履歴確認（業界情報）
```

### 2. 危険なアカウントの見分け方
```
【フォロー申請時のチェック項目】
❌ プロフィール画像なし・適当
❌ フォロー/フォロワーが極端に少ない
❌ アカウント作成が最近（1ヶ月以内）
❌ 投稿内容が不適切
❌ 同業他社への批判的リプライ
❌ 明らかな偽名・ニックネーム
❌ 過度に性的な内容への反応

➡️ 上記に該当する場合は慎重に判断
```

### 3. 施術前の最終確認
```
【当日のセーフティチェック】
📞 到着15分前の電話確認
🆔 ホテルロビーでの身分証確認
👥 同伴者がいないことの確認
🍺 飲酒していないことの確認
💊 体調・服薬状況の再確認
📱 緊急連絡先の再確認
```

## 💰 金銭トラブル防止

### 1. 料金回収の確実化
```
【支払い方法の工夫】
💵 現金事前支払いを基本とする
📱 電子マネー決済の導入検討
🏦 銀行振込（リピーター限定）
💳 クレジットカード決済（手数料考慮）

【未払い防止策】
📞 料金を事前に必ず確認
💰 施術前の料金受領
🧾 領収書の必発行
📋 支払い記録の詳細管理
```

### 2. キャンセル料金の管理
```
【キャンセルポリシー】
🟢 24時間前まで: 無料
🟡 24時間以内: 50%
🔴 当日・無断: 100%
🏥 体調不良: 無料（診断書で確認）
🌪️ 自然災害: 無料

【回収方法】
💳 事前デポジット制度
📞 電話での支払い約束
🏦 振込での後払い
❌ 次回利用停止措置
```
